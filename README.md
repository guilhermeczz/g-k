Uma experiência romântica interativa criada para transformar memórias especiais em uma pequena jornada.

O projeto acompanha **Karinha** por um caminho cheio de fases, perguntas e momentos marcantes. A cada resposta correta, um novo trecho da história é desbloqueado até chegar à surpresa final.

## Sobre a experiência

- Mapa romântico interativo.
- Bonequinha controlada por cliques.
- Cinco fases inspiradas em memórias do casal.
- Perguntas com validação e mensagens especiais.
- Caminho com animações, partículas e efeitos de luz.
- Sequência cinematográfica após o quiz.
- Personagens Karinha e Gui em pixel art.
- Portal animado e interação de manter o coração pressionado.
- Surpresa final com foto, música e animação neon K & G.
- Layout responsivo para celular e desktop.

## Tecnologias

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Executando localmente

Clone o repositório e instale as dependências:

```bash
git clone https://github.com/guilhermeczz/g-k.git
cd g-k
npm install
```

Inicie o ambiente de desenvolvimento:

```bash
npm run dev
```

Para gerar a versão de produção:

```bash
npm run build
```
